Projects

This section will outline some of the programming projects I've done over the past couple of years. Throughout these projects, the programming languages I used includes Javscript, HTML, CSS, Python, C, and Java! Click on the images if you want to see the Github repository with the code.

Project Description

Reversi: Apr 2024-Apr 2024

Using C, I developed and programmed the popular game of Reversi to be played between the user and the computer. Additionally, I programmed and developed an algorithm for the CPU that determines the best available move based on the moves made by the user. Validity and availability of user inputted moves were also taken into account.

Pacemaker Data Analyzer: Feb 2024-Feb 2024

At a hackathon, our team designed an application using Python that would be capable of analyzing data from a pacemaker, which in turn helps detect heart irregularities and the possibility of low battery levels in the pacemaker. Our team obtained pacemaker data from large datasets in order to analyze trends and get rid of possible outliers in our program. Additionally, we executed user interface testing to validate the app's usability and functionality of interactive elements on the application, such as buttons and menus.

Employee Management System: Feb 2023-Jun 2023

For my employee management system, I used Java to program back-end and Graphical User Interface for the application. I implemented a functional stack and queue system, using LIFO and FIFO principles, that was able to store the information of employees in a hash table data structure. System capabilities include adding, removing, and modifying employee profiles, as well as specifying employee attributes such as salary and experience.

President Card Game: Dec 2021-Jun 2022

I coded the popular card game President in Python. I designed and implemented game mechanics, such as shuffling cards, dealing cards, determining user turn order, and ranking players and cards. Additionally, I determined interactions between the cards placed by the players, and the outcomes of each card interaction in each scenario.
